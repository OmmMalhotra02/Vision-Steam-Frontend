## Features
- User authentication
- Channel creation and management
- Video browsing and playback
- Like, comment, and subscribe
- Playlist creation and management
- Channel dashboard with analytics
- Dark mode support
- Responsive UI

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Installation
- npm install
- npm run dev

## Environment Variables
VITE_BACKEND_URL

## Deployment
The frontend is deployed on Vercel and connected to the backend API.
